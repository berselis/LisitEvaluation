# App Registros
### Aplicación web mostrando información sobre registros de empleados utilizando base de datos en SQL Server.
- .Net Core 5.0 MVC
- SQL Server 2014
- HTML5
- CSS3
- Bootstrap v4.3.1
- DataTables.js V1.13.1
